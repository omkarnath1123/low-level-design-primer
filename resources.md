# Low-Level Design Resources

A curated list of books, blogs, and video channels to master object-oriented analysis, design patterns, and clean code principles.

## Book recommendations

* **Head First Object-Oriented Analysis and Design** – Great for learning how to analyze, design, and write serious object-oriented software.
* **Head First Design Patterns** – An excellent, highly visual introduction to standard design patterns.
* **Clean Code: A Handbook of Agile Software Craftsmanship** – Essential reading for writing readable, maintainable code.
* **Clean Architecture: A Craftsman's Guide to Software Structure and Design** – Focuses on software structure, boundaries, and components.
* **Refactoring: Improving the Design of Existing Code** – The definitive guide to improving code quality without breaking functionality.
* **Patterns of Enterprise Application Architecture** – Deeper patterns for building robust enterprise-level software.
* **Design Patterns: Elements of Reusable Object-Oriented Software** – The classic "Gang of Four" (GoF) book on design patterns.

## Blogs and interactive sites

* [Refactoring Guru](https://refactoring.guru/design-patterns) – Beautiful visual explanations of design patterns and refactoring.
* [SourceMaking](https://sourcemaking.com/) – Detailed guides on design patterns, anti-patterns, and UML diagrams.
